## Technology Stack

- **Frontend:** React 18, Vite, Tailwind CSS, react-icons
- **Backend:** Node.js, Express, Sequelize ORM
- **Database:** MySQL/MariaDB
- **Auth:** JWT (admin only, embed endpoints public)
- **Validation:** Client + Server side
- **Security:** CORS, Rate Limiting, XSS Protection

This architecture provides maximum flexibility while maintaining security and ease of use!
